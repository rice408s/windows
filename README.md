# RetroUI Generator

RetroUI Generator 是一个复古计算机界面模拟器，它可以模拟多种经典计算机和游戏机的用户界面。

## 支持的系统

- Windows 95
- MS-DOS
- Macintosh
- Amiga
- Atari ST
- Apple II
- ZX Spectrum
- MSX
- Game Boy
- Nokia 1100
- Commodore 64
- TRS-80
- BBC Micro

## 功能特点

- 精确还原各个系统的视觉风格
- 支持文本输入和编辑
- 支持主题切换
- 支持截图功能
- 模拟 CRT 显示器效果
- 保留原始系统的交互方式

## 开发技术

- React
- TypeScript
- Styled Components
- Vite
